# Netflix Clone

## About

This project is a simple Netflix homepage clone made using HTML and CSS.
It focuses on building a clean layout and understanding how real websites are structured.

## Features

* Netflix style homepage design
* Navigation bar
* Hero section with background
* Responsive layout
* Built only with HTML and CSS

## Technologies

HTML
CSS

## How to run

Download or clone the project and open the index.html file in your browser.

## What I learned

While building this project, I practiced:

* Structuring pages using HTML
* Styling with CSS
* Using Flexbox and Grid
* Making layouts responsive
